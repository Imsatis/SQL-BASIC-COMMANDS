<h3>#SQL-BASIC-COMMANDS</h3>

<HR>

<H1 align=center> BASIC COMMANDS FOR BEGINNERS </H1>

<HR>

##To Create Database..
```
 $ CREATE DATABASE *name; 
 ```

##To View Database..
```
$ SHOW DATABASE;
```

##To Use Database
```
$ USE *name;
```

##To Create Table..
```
$ CREATE TABLE *name (
    *name VARCHAR(SIZE),
    *name VARCHAR(SIZE),
    *name INT 
       );

```

##To Insert Data..
```
INSERT INTO *name ( *name, *name, [*name] )
   VALUES ( 'data','data',123 );

```
